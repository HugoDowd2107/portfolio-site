background

As a football fan there's so many games across Europe that it can be difficult to find the best ones to watch

Other football calendars focus on games happening that day so it's difficult to plan ahead. I wanted to build a website that surfaces important games so fans can plan ahead

I relied heavily on Claude Code and Figma Make to go from static design to prototype to production
